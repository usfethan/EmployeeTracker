# EmployeeTracker

## Description

A command line application to manage a company's employee database.

## Table of Contents
* Installation
* Usage
* License
* Tests
* Questions

## Installation
User should clone from githup repository.Requires are node.js-inquirer-console.table-mysql2.
To run app -> npm start.

## Usage
User will see all employees id department salary. Also user can update employees manager-salary-department. Also user can add new employee.
